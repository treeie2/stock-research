# 🔄 数据同步功能

## 功能说明

在个股详情页编辑数据后，可以通过同步功能导出、复制或清空编辑记录。

## 使用方法

### 1️⃣ 打开同步面板

1. 访问任意股票详情页
2. 在导航栏右侧找到 **「🔄 同步」** 按钮（青色）
3. 点击按钮打开同步面板

### 2️⃣ 查看编辑记录

同步面板显示：
- **总编辑数**：所有编辑操作的数量
- **涉及股票**：被编辑过的股票数量（去重）
- **最后编辑**：最近一次编辑的时间
- **编辑列表**：详细记录（倒序显示，最新的在前）

每条编辑记录包含：
- 时间戳
- 股票名称 + 代码
- 字段类型（⚡ 催化剂 / 💡 投资洞察）
- 内容预览（前 200 字符）

### 3️⃣ 导出数据

#### 方式一：下载 JSON 文件
1. 点击 **「📥 下载 JSON」** 按钮
2. 浏览器自动下载 `.json` 文件
3. 文件名格式：`edit_export_YYYYMMDD_HHMMSS.json`

**JSON 格式**：
```json
{
  "export_time": "2026-03-16T15:45:30.123456",
  "total_edits": 5,
  "edits": [
    {
      "timestamp": "2026-03-16T15:30:12.123456",
      "code": "300308",
      "name": "中际旭创",
      "field": "insights",
      "content": "中际旭创是算力板块中业绩兑现度最高的标的之一..."
    },
    ...
  ]
}
```

#### 方式二：复制内容
1. 点击 **「📋 复制内容」** 按钮
2. 内容自动复制到剪贴板
3. 可粘贴到邮件、微信、文档等

**复制格式**：
```
股票数据编辑同步 - 5 条更新
==================================================

时间：2026-03-16 15:30:12
股票：中际旭创 (300308)
字段：投资洞察
内容：中际旭创是算力板块中业绩兑现度最高的标的之一...

---

时间：2026-03-16 15:25:45
股票：新易盛 (300502)
字段：催化剂
内容：[1] 亚马逊、Meta 等硅谷巨头大幅上调 2026 年资本开支...

---
```

#### 方式三：清空记录
1. 点击 **「🗑️ 清空记录」** 按钮
2. 确认操作（防止误触）
3. 所有编辑记录被清空

**注意**：清空操作只清除编辑日志，不影响已保存的数据！

## 技术实现

### 后端 API

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/sync` | GET | 获取所有编辑记录 |
| `/api/sync/export` | GET | 导出为 JSON 文件下载 |
| `/api/sync/email` | POST | 生成邮件草稿（待实现） |
| `/api/sync/clear` | POST | 清空编辑记录 |

### 数据存储

- **文件位置**：`data/edit_log.json`
- **格式**：JSON 数组
- **自动保存**：每次编辑后自动追加记录

### 前端功能

- 同步面板（模态框）
- 实时加载编辑记录
- 统计信息展示
- 下载/复制/清空操作

## 使用场景

### 场景一：备份编辑记录
1. 编辑多只股票数据
2. 点击「同步」→「下载 JSON」
3. 保存到本地或云盘备份

### 场景二：汇报工作
1. 完成一批数据更新
2. 点击「同步」→「复制内容」
3. 粘贴到邮件/微信发送给团队

### 场景三：数据审计
1. 查看编辑历史记录
2. 确认哪些股票被修改过
3. 必要时清空记录重新开始

---

## 注意事项

1. **编辑记录独立于数据**：清空记录不会影响已保存的股票数据
2. **自动追加**：每次编辑自动追加到日志，不会覆盖
3. **时间戳**：使用服务器时间（Asia/Shanghai）
4. **内容截断**：日志中内容只显示前 200 字符（避免文件过大）
5. **导出完整**：下载的 JSON 文件包含完整内容（无截断）
