### 3. 全文搜索功能 (/search)

**已存在的搜索功能**，支持：

**搜索范围**:
| 字段 | 优先级 | 说明 |
|------|--------|------|
| 代码 | ⭐⭐⭐⭐⭐ | 精确匹配 (1000 分) |
| 名称 | ⭐⭐⭐⭐ | 包含匹配 (500 分) |
| 概念 | ⭐⭐⭐ | 概念标签匹配 (300 分) |
| 催化剂 | ⭐⭐ | 事件描述匹配 (200 分) |
| 投资洞察 | ⭐⭐ | 核心观点匹配 (200 分) |
| 公司概况 | ⭐⭐ | 核心业务描述 (200 分) |
| 行业地位 | ⭐ | 市场地位描述 (150 分) |
| 产业链 | ⭐ | 上下游位置 (150 分) |

**搜索示例**:
```

✅ 匹配投资洞察包含"AI"的股票
```
